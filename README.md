## ✨ 功能

- 🔍 实时搜索
- 🏷️ 分类筛选（格式转换 / 办公 / 音视频 / OCR / 设计 / 知识管理 / 效率工具）
- 📱 响应式设计，手机/平板/电脑适配
- 📲 支持安装为 PWA APP（离线可用）
- 🎨 深色科技风界面

## 🚀 部署到 GitHub Pages

1. Fork 或上传本仓库到 GitHub
2. 进入仓库 → Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后几分钟即可访问
