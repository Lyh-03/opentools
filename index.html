🧹 浏览器缓存清理

清理当前网站的浏览器缓存数据，包括本地存储、缓存文件、Service Worker 等。

💻 Windows 磁盘清理
1. 按 Win + R → 输入 cleanmgr
2. 选择 C: 盘 → 勾选要清理的项目 → 确定
📦 存储感知（自动清理）

Windows 10/11 自带功能，开启后自动定期清理临时文件和回收站。

📋 操作步骤：
1. 打开 设置（Win + I）
2. 点 系统 → 存储
3. 开启 存储感知
4. 点击可设置自动清理频率（每天/每周/每月）

💡 建议开启，自动保持磁盘空间

🗂️ 系统文件清理（深层清理）

清理 Windows 更新缓存、旧版本备份等系统文件，通常能清出几个G。

📋 操作步骤：
1. 按 Win + R → 输入 cleanmgr 回车
2. 选择 C: 盘 → 确定
3. 点左下角 「清理系统文件」 按钮
4. 勾选「Windows 更新清理」「以前的 Windows 安装」等
5. 点确定清理

⚠️ 清理「以前的 Windows 安装」后无法回退系统更新

⌨️ CMD 一键清理

用命令行快速清理临时文件，复制以下命令在管理员 CMD 中运行：

@echo off
echo 正在清理临时文件...
del /q/f/s %TEMP%\*
del /q/f/s C:\Windows\Temp\*
del /q/f/s C:\Windows\Prefetch\*
echo 清理完成！
pause

💡 保存为 .bat 文件 → 右键 → 以管理员身份运行
